# Network IP Class Checker & Subnetting Project

## 📌 Project Overview
This project is a combination of **Network Engineering** and **Web Development** concepts. It features a web-based IP checker that helps users identify the network class of a specific IP address.

## 🛠️ Technologies Used
* **HTML5:** For the web structure.
* **CSS3:** For styling and dark-mode UI.
* **JavaScript:** For the logic to analyze IP octets.
* **Cisco Packet Tracer:** Used to design a sample SOHO network topology.

## 🚀 Features
* Identifies IP Classes (Class A, B, and C).
* Responsive and user-friendly interface.
* Error handling for invalid IP formats.

## 📂 Project Structure
* `index.html` - The main structure.
* `style.css` - Custom styling.
* `script.js` - Logic for IP classification.
* `network-design.png` - Screenshot of the Cisco Packet Tracer design.

## 🧪 Testing
The project was tested using various IP ranges:
- **10.0.0.1** -> Identified as Class A
- **172.16.0.1** -> Identified as Class B
- **192.168.1.1** -> Identified as Class C
